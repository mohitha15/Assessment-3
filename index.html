• Simple Calculator
  • Array Operations
  • Object Manipulation

Clicking "Simple Calculator" will bring up a tool where you can perform basic arithmetic operations on two numbers and view the result. Selecting "Array Operations" will display a function that calculates and shows the sum of all even numbers within an array. Finally, clicking "Object Manipulation" will reveal a `Person` object with methods to retrieve the full name and increment the age, demonstrating object-oriented programming in JavaScript.
